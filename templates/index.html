O que é Python?

Python é uma das linguagens mais populares e versáteis do mundo, ideal tanto para iniciantes quanto profissionais experientes. Sua sintaxe simples e legível permite aprender programação de forma natural, enquanto sua versatilidade abre portas para áreas como desenvolvimento web, ciência de dados, inteligência artificial e automação.

Com alta demanda no mercado de trabalho e uma comunidade ativa e acolhedora, Python oferece excelentes oportunidades de carreira. Seu rico ecossistema de bibliotecas e frameworks acelera o desenvolvimento, tornando-o a escolha perfeita para quem busca produtividade e relevância no cenário tecnológico atual.

Estruturas de Seleção

As estruturas de seleção controlam o fluxo do programa baseado em condições. Em Python, utilizamos if, elif e else para criar decisões lógicas. Essas estruturas permitem que o programa execute diferentes blocos de código dependendo de condições verdadeiras ou falsas. A sintaxe é simples e usa indentação para delimitar os blocos de código. 

idade = 18
if idade >= 18:
    print("Maior de idade")
elif idade >= 16:
    print("Pode votar")
else:
    print("Menor de idade")
# Loop for
for i in range(5):
    print(f"Contando: {i}")

# Loop while
contador = 0
while contador < 3:
    print(f"Repetição {contador}")
    contador += 1

Estruturas de Repetição

Python oferece loops for e while para repetir operações. O loop for é ideal para iterar sobre sequências (listas, strings, ranges), enquanto while repete enquanto uma condição for verdadeira. Também temos comandos de controle como break (sair do loop) e continue (pular para próxima iteração).

Vetores e Matrizes

Em Python, trabalhamos principalmente com listas para representar vetores. Listas são estruturas dinâmicas que podem armazenar diferentes tipos de dados. Para matrizes, usamos listas aninhadas (lista de listas) ou bibliotecas como NumPy para operações matemáticas mais avançadas. As listas suportam indexação, fatiamento e diversos métodos de manipulação. 

# Lista (vetor)
frutas = ["maçã", "banana", "uva"]
print(frutas[0])  # Acessa primeiro elemento

# Matriz (lista de listas)
matriz = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
print(matriz[1][2])  # Linha 1, coluna 2
def saudacao(nome):
    return f"Olá, {nome}!"

def mostrar_info(nome, idade=25):  # Parâmetro padrão
    print(f"Nome: {nome}, Idade: {idade}")

# Chamando as funções
mensagem = saudacao("Maria")
print(mensagem)
mostrar_info("João", 30)

Funções e Procedimentos

Funções em Python são definidas com def e podem receber parâmetros e retornar valores. Elas promovem reutilização de código e modularização. Python não distingue formalmente entre funções e procedimentos - uma função que não retorna valor explicitamente retorna None. Suportam parâmetros padrão, argumentos nomeados e quantidade variável de parâmetros.

Tratamento de Exceções

O tratamento de exceções usa blocos try, except e finally para capturar e gerenciar erros durante a execução. Isso permite que o programa continue funcionando mesmo quando ocorrem problemas, oferecendo uma forma elegante de lidar com situações inesperadas como divisão por zero, arquivos inexistentes ou dados inválidos. 

try:
    numero = int(input("Digite um número: "))
    resultado = 10 / numero
    print(f"Resultado: {resultado}")
except ValueError:
    print("Erro: Digite apenas números")
except ZeroDivisionError:
    print("Erro: Não é possível dividir por zero")
finally:
    print("Operação finalizada")
